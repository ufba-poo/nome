# Exercício de POO

Para saber o que deve ser feito no exercício, leia o código nos arquivos `.java` disponíveis em `src/main/java` e edite os arquivos de acordo com as orientações nos comentários de código.

Use boas práticas de programação orientada a objetos em Java.

Para verificar se o seu código está correto, execute o código de testes disponível na pasta `src/test/java`.

## Política para resolução dos exercícios para fins de avaliação

Se você está fazendo este exercício como parte de uma avaliação, você deve obedecer as orientações a seguir.

- **Autoria individual**. Todos os exercícios devem ser realizados individualmente pelo estudante. Ao submeter uma atividade, o estudante declara que o trabalho é de sua própria autoria e que foi desenvolvido de forma independente.
- **Consulta a outras pessoas**. Os professores e monitores poderão ser consultados para dirimir dúvidas sobre o exercício. Outras pessoas também podem ser consultadas, com moderação, mas estas não devem produzir código para a solução do exercício, se limitando a discutir conceitos e estratégias gerais.
- **Uso de materiais de apoio**. É permitido o uso de livros, anotações, materiais didáticos fornecidos pelo professor, bem como outras fontes tradicionais de estudo, desde que devidamente compreendidas e utilizadas como apoio ao aprendizado.
- **Proibição do uso de Inteligência Artificial**. Não é permitido o uso de ferramentas de Inteligência Artificial (como assistentes de código, geradores de texto ou similares) para gerar respostas completas, gerar partes das respostas, ou sugerir soluções que sejam incorporadas ao trabalho final.
- **Declaração de autoria do código**. Ao entregar exercícios que envolvam programação, o estudante declara explicitamente que todo o código foi escrito por ele próprio, sem auxílio de ferramentas automatizadas de geração de código, com exceção de geradores de código simples disponíveis nas IDEs.
- **Integridade acadêmica**. O descumprimento desta política será considerado violação de integridade acadêmica e estará sujeito às medidas previstas pelas normas da instituição e da disciplina.
